# My Cli

## About

Just a simple CLI

## Setup
To start the project will be necessary run the following scripts:
```bash
#Clone the repository
$ git clone https://github.com/ravinmor/my-cli

#Access the backend app folder
$ cd my-cli

#Install packages
$ yarn

#Start the app
$ yarn start

```
